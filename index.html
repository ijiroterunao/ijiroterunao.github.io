未命名

发表于 2018-05-27 | 分类于 随笔

事实是人类世界发展到今天，是靠人文科学和自然科学两条腿共同前进的。
创新并不是只要努力，只要砸钱就可以出来的，人文科学的进步也是很重要的。
技术创新一定需要资金的支持，瓦特发明蒸汽机的之前没人愿意给他钱，需要有专利保护，需要有完成交易的金融信誉制度，需要有资本主义提供的这一切社会制度和关系，否则他一辈子都干不了蒸汽机。虽然蒸汽机成功地推动了工业革命，释放了资本主义巨大的生产力，但是技术变革本身能够出现，他是有赖于资本主义提供的制度环境的。我们换个角度想，就算瓦特从来没有出生过，但是资本主义的繁荣客观上就需要运输设备和生产设备能有更高的动力，能带来更大的生产效率，而不断发达的社会分工，它必然会催生出一批专业的技师和机械师，就算他们不走瓦特的路，也总有别的路可走，也就是需求导致了技术进步。
亚当斯密的国富论首次给资本主义描绘了一幅清晰的世界地图，从基础上建立起对国家经济运行的一个正确的认识，从而政府才能理性的设计自己的财政金融政策，才知道怎样管理经济活动让国家富强，而那些基于经济的远期投资活动，它也要有这种理论指导。
不要一提资本主义就想到西方，中国现在的制度，至少在经济上还是用的资本主义的那一套的，但是任然不完善，还需要再改进，创造符合中国特色的制度。

理论源于实践，而后才是理论引导实践，但最终，理论还要靠实践来修正。本身是鸡生蛋、蛋生鸡的关系，对理论和实践不同领域取得的成就，何苦去排序呢，终究是人文观念过重。

个人与环境

发表于 2018-05-27 | 分类于 随笔

记得语文老师说人应该自己努力来摆脱环境的控制。然后举了几个靠自己努力改变了“命运”的例子。
我以前觉得挺有道理，即使环境差，努力了就能成功不是吗？
（我这里讲的环境是包含一切外在因素。）
但是我发现，环境和个人是类似于鸡和蛋的关系，是环境导致了个人的成就还是个人的性格改变了自己最后呆的环境，它们相互纠缠。说光靠自己努力就能摆脱命运，我觉得是可笑的，因为就连你自己的性格都是环境给你的。当然你的性格可以再改变环境，但是这种改变的根源不还是环境吗。结论是当然要熟练掌握投胎技巧啦。🐽

不过我觉得语文老师的意思大概是环境已经是这样了，你也改变不了，只能改变自己。那么我的回答和他的回答已经是两个问题了，没啥好讨论的。

发表于 2018-03-12 | 分类于 随笔

lspci | grep -i vga

显示机器上的显卡信息

[prac_18@mu01 ~]$ lspci | grep -i vga
0a:00.0 VGA compatible controller: ASPEED Technology, Inc. ASPEED Graphics Family (rev 30)                                          
[prac_18@mu01 ~]$

如果想看某个显卡的详细的信息，比如0a:00.0

[root@localhost conf]# lspci -v -s 02:00.0
